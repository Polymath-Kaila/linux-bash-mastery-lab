# 30-Day Linux + Bash Roadmap

## Week 1: Linux Fundamentals

Goal: Become comfortable moving around the terminal.

Topics:
- Filesystem structure
- Navigation
- File and directory management
- Reading files
- Help and man pages
- Permissions

Deliverables:
- Basic command notes
- Permission notes
- Navigation exercises

## Week 2: Command Line Power

Goal: Use Linux tools together.

Topics:
- Pipes
- Redirection
- grep
- find
- sort, uniq, wc, cut, tr
- Processes
- Archives and compression

Deliverables:
- grep examples
- find examples
- text-processing exercises

## Week 3: Bash Scripting

Goal: Write useful Bash scripts.

Topics:
- Shebang
- Variables
- Arguments
- Exit codes
- Conditionals
- Loops
- Functions
- Error handling

Deliverables:
- backup script
- system info script
- log analyzer script

## Week 4: System Admin Basics

Goal: Understand practical Linux operations.

Topics:
- Users and groups
- Package management
- systemctl
- journalctl
- cron
- networking commands

Deliverables:
- cron examples
- service notes
- final mini-projects
