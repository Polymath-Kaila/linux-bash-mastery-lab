# Linux Bash Mastery Lab

A structured 30-day learning repository for mastering Linux command line, Bash scripting, system administration basics, and practical automation.

## Goal

To become confident using Linux and Bash by learning through notes, commands, exercises, scripts, debugging, and mini-projects.

## Learning Areas

- Linux filesystem and navigation
- Files, directories, permissions, and users
- Pipes, redirection, grep, find, and text processing
- Bash scripting fundamentals
- Processes, services, logs, cron, and networking
- Practical automation projects

## Repo Structure

| Folder | Purpose |
|---|---|
| `00-setup` | Environment setup notes |
| `01-linux-fundamentals` | Core Linux command-line basics |
| `02-command-line-power` | Pipes, search, text processing, processes |
| `03-bash-scripting` | Bash notes, scripts, and exercises |
| `04-linux-system-admin` | Admin basics: users, packages, services, logs, cron |
| `05-mini-projects` | Practical automation projects |
| `06-quizzes` | Weekly self-tests |
| `07-errors-and-debugging` | Mistakes, errors, and fixes |
| `08-reference` | Cheat sheets and reference notes |
| `sandbox` | Safe practice area |

## Study Method

1. Learn the concept.
2. Write notes.
3. Run commands.
4. Break something safely in `sandbox`.
5. Fix it.
6. Document the lesson.
7. Build a script or mini-project.
